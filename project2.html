Project 2

This is an application that lets users search up information about Pokemon. The information included are the Pokemon's name, type(s), pokedex entry number, pokedex description, height & weight, the category of the pokemon, and its abilities. The information is pulled from pokeapi, using the /pokemon and /pokemon-species endpoints

This application was created using React + Vite and CSS
